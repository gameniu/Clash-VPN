## Clash机场 / Clash节点订阅使用教程（新手必看）

目前主流翻墙机场梯子基本都推荐搭配 **Clash机场 + Clash节点订阅** 使用，稳定、省心。

### 使用步骤

1. 在机场官网获取 **Clash订阅链接**
2. 安装 Clash for Windows / Clash Verge / Shadowrocket / Stash 等客户端
3. 在客户端中粘贴订阅链接并更新配置
4. 启用 **规则模式（Rule）**，选择延迟低的节点即可使用

这种方式可以自动更新机场节点，是目前最稳定的 **外网梯子** 使用方案之一。
